# How to set up environment

## Set `.env`

- Create `.env`.
```
toucth .env
```
- Set the following environment variable.
```
SLACK_BOT_TOKEN=<yours-token>
SLACK_SIGNING_SECRET=<yours-sigining-secret>
```

## Install package

- install node packages.
```
npm install
```

## Run nodejs

```
node app.js
```
